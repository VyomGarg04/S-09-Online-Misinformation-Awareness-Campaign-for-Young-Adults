# S-09: Online Misinformation Awareness Campaign for Young Adults

## Social Challenge
**Misinformation — health myths, fake news, manipulated media — spreads rapidly through WhatsApp and social media among young adults, causing real-world harm (vaccine hesitancy, communal tension, financial scams).**

## Background
Media literacy in the digital era is crucial. A combination of awareness campaigns and interactive workshops helps build the necessary critical thinking skills for young adults to verify information before sharing.

## Project Objectives
1. Assess baseline media literacy and misinformation susceptibility in the target group
2. Design a multi-format misinformation awareness campaign (digital + offline)
3. Conduct workshops on fact-checking tools and critical media evaluation
4. Create an open-access fact-checking resource guide
5. Partner with student media clubs or journalism programs
6. Measure improvement in information verification behavior post-campaign

## Stakeholders
| Stakeholder | Role |
| :--- | :--- |
| University students (18-25) | Primary audience |
| Local college youth | Secondary audience |
| Student journalism / media clubs | Content and distribution partner |
| Faculty media/communication department | Advisor |
| Fact-checking organizations (Boom, Alt News) | Content validation partner |

## Requirements
- **Research**: Review misinformation typology (satire, fabrication, manipulated content); study existing media literacy frameworks (SIFT, CRAAP, UNESCO MIL).
- **Survey**: Misinformation susceptibility scale; fact-checking behavior frequency; platform usage habits; post-campaign same instrument.
- **Execution**: Campaign (posters, reels, WhatsApp stickers), 2 workshops (fact-checking tools: Reverse Image Search, Snopes, BOOM), resource guide creation.
- **Documentation**: Survey data, campaign reach metrics (social media analytics), workshop attendance, resource guide.
