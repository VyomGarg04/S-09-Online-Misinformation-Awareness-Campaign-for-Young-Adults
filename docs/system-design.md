# MediaShield Domain Model

## Core Entities

### User

Represents a registered platform user.

### Article

Educational content related to misinformation and digital literacy.

### Resource

External fact-checking websites, tools, and references.

### Quiz

Interactive learning assessment.

### Question

Individual quiz questions.

### Survey

Media literacy surveys conducted before and after campaigns.

### Survey Response

Stores participant responses.

### Claim Verification

Stores AI-assisted claim verification requests.

### Verification History

History of claims verified by users.
